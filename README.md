# What is Rezzy Legends?

Rezzy Legends is a standalone **Dragon Ball Legends-style summon simulator** designed to recreate the excitement of chasing rare units and building your collection.

Summon your way through multiple banners, unlock rewards, track your progress, and experience a custom DBL-inspired sandbox built from the ground up.

Pull from:

- LEGEND Units
- ULTRA Units
- LL Units
- SPARKING Units
- EXTREME Units
- HERO Units
- Future custom rarities

---

# Current Features

### Summon System
- Single and multi summons (1x / 10x)
- Multi-banner support
- Fake-out summon moments
- Rarity glow effects
- Auto-layout summon results
- Summon history tracking

### Profile System
- XP and leveling
- Player statistics
- Rare pull tracking
- Total CC spent
- Future profile customization

### Missions & Rewards
- Daily missions
- Weekly missions
- Monthly missions
- Long-term achievements
- CC rewards
- Item rewards

### Social Features
- Friends system
- Profile progression
- Main menu hub

### Future Plans
- Custom events
- Custom banners
- Character creator support
- DBL-style summon animations
- Clans
- PvP (experimental possibility)
- More events and game modes
- Additional rarities
- More surprises
