## What is Rezzy Legends?

Rezzy Legends is a standalone **Dragon Ball Legends-style summon simulator** designed to recreate the excitement of pulling rare characters and chasing insane summons.

Experience a custom DBL-inspired sandbox featuring banners, progression systems, rewards, profiles, and future online features.

Pull from:

* LEGEND Units
* ULTRA Units
* LL Units
* SPARKING Units
* EXTREME Units
* HERO Units
* Future custom rarities

---

## Current Features

### Summon System

* Single summons (1x)
* Multi summons (10x)
* Multi-banner support
* Fake-out summon moments
* Rarity glow effects
* Auto-layout summon results
* Summon history tracking
* Future summon animations

### Profile System

* XP & leveling
* Total CC spent
* Rare pull tracking
* User statistics
* Future profile customization
* Titles & collectibles planned

### Social Features

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Rez-Yeat/RezzyLegends.git
cd RezzyLegends
```

### Open Project

Open the project in Android Studio and run:

```bash
./gradlew build
```
